# Важно знать перед использованием

Расширение ругается на прямые пути по /src/cf, /src/cfe и т.д, если папки не пустые и там остаются, например, README.md.
Вы можете их спокойно удалить, если прочитали информацию и разобрались. 

Или создать внутри ещё пустую папку/директорию с файлами, но тогда не забудьте поменять пути к ним в env.json
